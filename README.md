# AWESOME ALGEBRA
대수학은 정말 어~썸할 것이다

## 저장소의 정체
추상대수학, 선형대수학의 개인적인 학습 내용을 정리하는 저장소이다.

## 학습 이유
~~현대수학에 추상대수학 관련내용만 나오면 알아쳐먹기가 힘든게 너무 꼬와서~~
수학에 대한 순수한 지적 호기심

## 학습 목표
추상대수학 관점으로 선형대수학을 이해하는것. 최종적으로 텐서에 대한 대수학적인 이해를 목표로 한다.

## 저장소 구성
- 학습목표를 바탕으로 LLM 과 소통하며 학습목표와 목차를 구성했다.
- 추후 정적사이트 배포를 고려하여 파일구조를 구성했다
- part, chapter, section 으로 구성되어있으며 section 이 단일 파일로 구성된다
- 목차는 나중에 정적사이트 생성기가 만들어주겠지?

## 학습 방법
- LLM 과 질의를 통해 내용에 대한 정리를 하고 이해한 내용을 **직접** 정리해서 내용을 구성한다.
- 눈으로만 보고 이해하면 제대로 익히지 못한다. 백지에 직접 증명하고 검증한 내용을 남긴다.
- 학습 이후에 복습을 원활하게 하기 위해 학습하며 생긴 의문점들과 제대로 이해하면 풀 수 있는 연습문제를 반드시 남겨야 한다.
